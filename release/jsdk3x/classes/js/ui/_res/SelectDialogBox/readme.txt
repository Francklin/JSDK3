功能点描述：

1.支持IE6+、Firefox4+、Chrome13+、Safari5+、Opera11+、Mobile Safari
2.两种界面，分别支持简单、标准、高级三种选择模式
3.第一种界面全部使用选择框设计，分别对适应所有浏览器的标准模式、IE的兼容模式和iPad Safari等三种情况做了页面处理。
4.第二种界面是视图区采用树形菜单显示，也分别对浏览器的三种情况做了页面处理

关于选择模式：
1.简单选择模式
没有分类、没有子数据，只能对单层列表值进行单选、复选
2.标准选择模式
没有分类，但可以有子数据，也只能对单层列表值进行单选、复选
3.高级选择模式
函盖所有功能，支持树形结构菜单显示，可以有分类，也可以有子数据，并且
视图区的数据也可以是多层结构

JSDK3 (JavaScript Development Kit)

Latest release: v1.9.1 beta 20140814, start at 2010, since 2007

Support Browsers:

Features:

  • support multiple browser: IE6+, IE9+, Firefox 3.6+, Chrome 13+, Safari 5+, Opera 11+
  • support multiple language
  • support multiple version of class coexistence
  • support the namespace management for class
  • support dynamic loading class
  • support powerful DOM template
  • encapsulates a part of the basic prototype objects of javascript: String, Number, Date
  • encapsulates a part of DOM API of browser
  • contains lite edition
